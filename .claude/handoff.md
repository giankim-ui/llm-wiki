현재 세션에서 진행한 작업을 바탕으로, 현재 작업 디렉토리에 핸드오프 파일을 작성해줘.

**파일명 규칙**: 현재 작업 디렉토리에 이미 같은 이름의 파일이 있으면 기존 파일은 그대로 두고, 새 파일은 번호를 붙여 저장해:
- `HANDOFF.md` 없으면 → `HANDOFF.md`
- `HANDOFF.md` 있으면 → `HANDOFF-1.md`
- `HANDOFF-1.md`도 있으면 → `HANDOFF-2.md`
- 이후 동일하게 숫자를 1씩 올려 중복되지 않는 이름으로 저장

다음 에이전트가 이 파일 하나만 읽고 바로 이어갈 수 있도록, 아래 4개 섹션을 명확하게 작성해:

## 시도한 것
- 이번 세션에서 시도한 모든 작업을 항목별로 나열 (성공 여부 무관)

## 성공한 것
- 완료되었거나 확인된 결과물 목록 (파일 경로, 변경 사항 포함)

## 실패한 것 / 막힌 것
- 실패하거나 해결되지 않은 항목, 오류 메시지나 원인 포함

## 다음 단계
- 다음 에이전트가 바로 이어서 해야 할 작업 목록 (우선순위 순)
- 필요한 파일 경로, 컨텍스트, 주의사항 포함
