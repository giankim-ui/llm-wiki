# **1.TASK 이동 기능 개선**
기존 KR.NO의 순서에 따라 D&D 순서를 바꿀수 없는 구조를 개선
예시 : KR3. 인사 내 정형, 비정형데이터의 기존데이터 DB 구축 도 KR1. AI 활용으로 운영 인력 효율성 강화 위로 옮길 수 있도록 함

# **2.TASK 완료 내역 별도 탭 신설**
웹 앱 중간부분 TASK section 중 생성된 TASK 행 중에서 개별 TASK의 시작일, 기한, 완료 체크가 된 항목은 자동으로 완료 탭으로 이동
이를 위해 '완료' 탭을 신설
기존 내용은 '작업중' 탭만 새로 만들어 유지
완료의 트리거는 TASK 단위 내 모든 작업이 완료되었을 때
모든 작업 완료의 뜻은 시작일, 기한이 입력되어 있고 완료 체크박스가 클릭 되었을 경우를 말함
(Don't do it) 
(1) TASK 내 여러 SUB-TASK 중 일부만 완료되었을 때는 완료 탭으로 이동하지 않음
(2) 완료가 되었어도 상단 세션인 OKR에서의 진행율은 기존 계산과 같아. 즉 '작업중' + '완료' 의 합계로 계산됨('작업중'과 '완료' 각 각 계산하면 안됨)
