## 1. 초안 아웃룩 임시보관함 자동 이동 기능
 -내 MS 아웃룻 환경은 M365야. 현재 mailing-agent가 작업한 결과물 초안을 자동으로 내 아웃룩 내 임시보관함에 저장하는 기능을 추가해줘.
 -해당 내용을 첫 개발로 하고 phase를 구분해줘.
 -코딩 후 반드시 시험 작동 여부를 먼저 사용자에게 테스트 해줘.

## 3. /draft 이후 질문단순화
'commnads/draft.md' 파일을 수정했어 내가 수정한 부분을 파악하고 분석해서 반영해줘.


## 4. 표 기능 개선
아웃룩은 html 붙여넣기를 미지원하나 실무에서는 마크다운 형태의 표는 사용하지 않아. 표 형식 정리를 npm 설치를 통해 지원할수 있는 방법이 있다면 지원해줘.(research 단계로 진행)
