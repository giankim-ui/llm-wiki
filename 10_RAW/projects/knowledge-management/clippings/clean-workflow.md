# 요청사항
- 현재 상황에서 가장 효율적으로 바로 직전 업무 및 찾고자 하는 직전 프로젝트 마크다운 문서를 찾을 수 있는 구조를 제안해줘.

# 상황
- 현재 claude code cli 환경에서 antigravity을 ide로 활용하고 있어. 
- 나는 ai와 협력하여 research.md - plan.md - execution 방식으로 일을 하고 있어
- /!claude 폴더 아래 각 작업시 새로 폴더를 생성하여 프로젝트를 진행 하고 있어.
- 각 마크다운 파일은 프로젝트 별 archive 파일에 저장하고 있어.
- v#.# 등 버전을 기록해 관리중이야.

# 목적
각종 업무를 코딩 및 AI를 활용하여 자동화하고 HR 데이터 분석을 통해 인사이트를 얻고자 하는게 사용목적이야.

# 문제점 
- 여러 프로젝트를 개발하고 수정하면서 ideation 하기 때문에 최근 내가 했던 작업이 뭐였는지 찾는데 시간이 오래 걸려
- 비슷하게 최근했던 작업이 정리되어 있는 마크다운 파일을 찾는 것도 헤매느라 비효율적이야
